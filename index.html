About Me

My journey began with Java, where I honed my skills in building scalable and efficient applications using Spring Boot. Alongside, I have ventured into the realm of web development, mastering JavaScript, ReactJS and React Native to create captivating and responsive user interfaces.

My expertise extends to the core of web development, as I seamlessly blend HTML and CSS to shape delightful and intuitive digital experiences. The realm of databases has always intrigued me, leading me to become proficient in managing data across PostgreSQL, MySQL and MongoDB platforms.

I believe in the power of the cloud to drive modern software development, which is why I've embraced AWS services to build and deploy applications that can effortlessly scale. Additionally, I'm well-versed in streamlining communication through Apache Kafka, and I have hands-on experience orchestrating containerized applications using Docker and Kubernetes.

Join me on this journey as we navigate the dynamic world of software engineering together, fueled by curiosity, collaboration and the pursuit of excellence.
